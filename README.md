
### Build Docker 
```
$ docker compose -f docker-compose.dec.yml up -d
```

### Activate Virtual Environment

- **Windows**:

    ```bash
    ./venv/Scripts/activate
    ```

- **Linux/Mac**:

    ```bash
    source ./venv/bin/activate
    ```
